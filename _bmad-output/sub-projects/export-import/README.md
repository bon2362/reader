# Export/Import Feature

Эта папка собирает в одном месте все артефакты по фиче экспорта и импорта аннотаций в Markdown.

## Содержимое

- `project-docs/` — архитектурные и planning-документы фичи
- `stories/` — BMad story-файлы эпика

## Канонические источники

Чтобы не ломать существующие BMad-пути и workflow, исходные файлы продолжают жить в стандартных местах:

- `_bmad-output/project-docs`
- `_bmad-output/stories`

В этой папке лежат копии для удобной навигации по фиче как по одному пакету.
