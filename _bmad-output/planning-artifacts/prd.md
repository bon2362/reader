# Product Requirements Document: Reader App

**Версия:** 1.0 | **Дата:** 18.04.2026
**Статус:** Утверждён

---

## Обзор продукта

Персональный EPUB/PDF читатель для Mac с глубокой системой margin-аннотаций. Приложение для себя, не для App Store. Фокус: качество чтения + надёжное хранение заметок.

**Стек (зафиксировано):**
- Платформа: macOS (Swift + SwiftUI)
- EPUB движок: Readium Swift Toolkit
- Хранение: SQLite локально
- Будущее: iPhone (тот же Readium toolkit), iCloud sync

---

## MVP — EPUB Reader для Mac

### F-01: Библиотека
- Импорт EPUB через стандартный file picker (файл копируется в app sandbox)
- Список книг: обложка, название, автор, прогресс (%)
- Последняя открытая позиция сохраняется и восстанавливается
- Удаление книги из библиотеки

### F-02: Reading View
- Постраничный рендеринг через Readium (paginated mode)
- Навигация: стрелки, клавиши ←/→, клик по краям экрана
- Тулбар: скрывается при чтении, появляется по движению мыши
- Тулбар содержит: название книги + название текущей главы
- Нижняя строка: номер страницы / всего страниц

### F-03: Оглавление
- Боковая панель с иерархическим TOC из EPUB metadata
- Переход к главе по клику
- Текущая глава подсвечена

### F-04: Поиск по тексту
- Cmd+F → строка поиска
- Подсветка всех вхождений, переход prev/next
- Поиск по всей книге

### F-05: Highlights
- Выделить текст → контекстное меню → выбрать цвет
- 5 цветов: жёлтый, красный, зелёный, синий, фиолетовый
- Клик по хайлайту → меню: изменить цвет / удалить
- Привязка: EPUB CFI (стандартный якорь)

### F-06: Аннотации (два типа)

**Тип A — Комментарий к тексту:**
- Выделить текст → "Add note" → текстовое поле в правом margin
- Иконка-маркер на поле напротив соответствующего абзаца
- Клик по маркеру → разворачивает/сворачивает текст заметки
- Может существовать без highlight (просто к месту в тексте)

**Тип B — Sticky note к странице:**
- Кнопка в тулбаре или Cmd+Shift+N → создать заметку к текущей странице
- Иконка стикера в правом margin, не привязана к конкретному тексту
- Клик → открывает/сворачивает

### F-07: Панель аннотаций
- Боковая панель, 4 вкладки: All / Highlights / Notes / Sticky Notes
- Каждый элемент: цитата или превью текста, глава, страница
- Клик → переход к странице
- Сортировка: по порядку в книге (дефолт)

### F-08: Хранение
- SQLite база, сохранение мгновенное при каждом действии
- Книги: метаданные + путь к файлу в sandbox
- Аннотации: CFI-якорь, тип, цвет, текст, timestamp, book_id
- Sticky notes: page_index, текст, timestamp, book_id

## Roadmap

### Фаза 2 — Поиск и экспорт аннотаций
- Поиск по тексту хайлайтов и заметок
- Фильтр хайлайтов по цвету
- Теги для аннотаций
- Экспорт в Markdown/JSON с контекстом (глава, страница, цитата)

### Фаза 3 — PDF
- PDFKit рендеринг (нативный macOS)
- Highlights и заметки в PDF (координатная привязка)
- Общий UI для EPUB и PDF

### Фаза 4 — Sync и резервирование
- Ручной экспорт/импорт SQLite (backup в папку / iCloud Drive)
- iCloud sync через CloudKit
- iPhone-версия на том же Readium toolkit

### Фаза 5 — Power features
- Статистика чтения (время, страниц в день)
- Smart-коллекции (по прогрессу, плотности аннотаций)
- Интеграция с Obsidian / Notion

---

## Зафиксированные решения

| Решение | Выбор |
|---------|-------|
| EPUB движок | Readium Swift Toolkit |
| Аннотации | Margin-подход (текст не перекрывается) |
| Привязка аннотаций | EPUB CFI |
| Хранение | SQLite локально, мгновенное сохранение |
| Файлы книг | Копируются в app sandbox |
| Стек | Swift + SwiftUI + Readium |
| Изменения файлов | Не предполагаются, CFI не пересчитывается |
